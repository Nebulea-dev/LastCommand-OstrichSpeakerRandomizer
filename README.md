# MelonLoader Last Command OstrichSpeaker Randomizer Mod
This mod randomizes the computations and adds trigonometry questions to the boss OstrichSpeaker in Last Command.

# Features
- Randomizes math questions
- Adds trigonometry questions (cosine, sine, double angle, sum/difference formulas)
- Works with all difficulty levels

# Installation
1. Install MelonLoader for Last Command

Download MelonLoader from the official website: MelonLoader Installer
Run the installer and select Last Command as the target game.
Follow the on-screen instructions to complete the installation.

2. Install this Mod

Download the latest release of this mod from the [Releases page](https://github.com/Nebulea-dev/LastCommand-OstrichSpeakerRandomizer/releases/tag/1.0.0).
Copy the MelonLoader_LastCommand_OstrichSpeaker.dll file into your Last Command Mods folder (likely at C:\Program Files (x86)\Steam\steamapps\common\Last Command\Mods).

3. Launch the Game

Start Last Command as usual through Steam.
The mod will be active and will randomize OstrichSpeaker questions.
